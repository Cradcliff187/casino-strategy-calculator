# Casino Strategy Calculator

A premium, lightning-fast casino strategy calculator for 3 Card Poker, Blackjack, and Video Poker (Jacks or Better).

## Features

- **Visual Card Selection** - Click cards instead of dropdowns (50% faster!)
- **Keyboard Shortcuts** - Type "KH" for King of Hearts, Space to confirm
- **Real-time Results** - Instant optimal play decisions
- **Recent Hands History** - Quick reload of last 3 hands
- **Copy to Clipboard** - Share results with one click
- **Mobile Responsive** - Touch-friendly design with horizontal scrolling
- **Professional UI** - Casino-themed design with smooth animations

## Games Supported

- **3 Card Poker** - Q-6-4 strategy (3.37% house edge)
- **Blackjack** - Basic strategy (0.5% house edge)
- **Video Poker** - Jacks or Better optimal play (99.54% RTP)

## Getting Started

### Development

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

This app is configured for Netlify deployment. The `netlify.toml` file includes:
- Build command: `npm run build`
- Publish directory: `dist`
- SPA redirect rules
